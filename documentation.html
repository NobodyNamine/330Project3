Documentation

Process:

I decided to tackle the anime API. My idea was to copy paste gif finder and build off that. Then I looked at the documentation for the anime API to realize it didn't need an API Key. I also looked at how to get information from that API. I had my prototype ready where it was taking in a genre and returning at least 10 anime from that genre.

I improved upon the controls to take the score into account. I worked on adding a flexbox system to my drop downs. Then I updated the status symbol/text, took limit into account,and had the results display a score. Somewhere along the line I started working on sorting the data by score instead of it's default: alphabetical. An hour or so later I realized that using the sort function wasn't going to work so I used a bubble sort to format my information.

My final website design strayed far from my prototype as I really liked the idea of having the status image be an anime character instead of a circle. The pallete was more or less the same. I added a footer in for the documentation and kept everything center aligned.

Sources:

All the images were labeled for noncommercial reuse with modification. Except the anime character art which was supplied by an artist, Ezaki Bisuko, the character is called Menhera-chan. The other source is the Jikan Unofficial MyAnimeList API.

Things to Notice:

Something is wrong on the API end when it comes to getting the score. In the higher ranges, if you select 7 it will only give you scores below 8 which is correct, but if you select 3 it will give you all the score it has for that category starting from the lowest. I checked this on the MyAnimeList Website itself with the same results.

If you select a combination with no results it will show a special image. Try something that would have no results like Dementia, 9, and 5. I added some css responsiveness (flexbox and grid) to my results and made my link and button fancy. I added a favicon and implemented smooth scrolling.

Grade:

I made sure to review the rubric and checked it with the website. Based on the rubric I would give myself an A. I am overall very pleased and proud of the final result of this website. I'm not sure what I could have added to go above and beyond, but there's always room for improvement with anything.
